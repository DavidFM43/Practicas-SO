# **Description**

This program manages Uber-Movement trips information, specifically, it provides a CLI for the user to query the average time of a trip based on certain basic trip criteria.
## **Usage**
1. Process data.
```bash
make data
```
2. Launch server. 
```bash
make run_server
```

2. Launch client. 
```bash
make run_client
```
